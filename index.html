Colorblind drawings

The colorblind will only see circles

About

This tool provides the user a drawing canvas for then generating an image composed by circles colored in a certain way to be hard for a colorblind to distinguish.

Create the image that you want to hide from colorblind people

Draw your image inside the box

Use the white color to erase and then select black to continue drawing.
Also you can clear the whole box with the 'clear' button.

Set the generator configuration

Choose a color palette

Note that some people can distinguish some of the following color schemes but will struggle with others.

Set the radius of the circles

This is for the radius of the circles that will fill up the drawing. They are generated choosing
a random number between the min and max values. Note that with a bigger circle the resolution
will be lower and with a smaller radius will take more time to generate.
